Every element can be traversed in an array using a single loop. Arrays make sorting much easier. Elements can be sorted by writing a few lines of code. loop

The body of do...while loop is executed once. Only then, the testExpression is evaluated. If testExpression is true, the body of the loop is executed again and testExpression is evaluated once more. This process goes on until testExpression becomes false. If testExpression is false, the loop ends.
